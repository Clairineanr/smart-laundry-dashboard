# SmartLaundry Dashboard

Business Intelligence Dashboard for monitoring laundry operations, revenue performance, customer analytics, and logistics activity.

## Features

* Executive KPI Dashboard
* Revenue Analytics
* Customer Intelligence
* Demand Forecasting
* AI Insights
* Driver Performance Monitoring
* Laundry Partner Analytics
* Logistics Operations Overview

## Tech Stack

* Next.js 16
* TypeScript
* Tailwind CSS
* Recharts
* Lucide React
* Vercel
